# Getting started

## Installation

Inside your project directory, run

```
yarn add @kaito-http/core
```

Or with npm:

```
yarn add @kaito-http/core
```

## Quick start

Kaito is simple to use. From the ground up, we left it fairly unopinionated to _let your creativity run wild_.

## Basic Example

```ts
import { Kaito, Controller, Get, Post, Schema, KRQ, KRT } from "@kaito-http/core";

@Controller("/")
class Home {
  @Get("/")
  async home(): KRT {
    return { body: { success: true } };
  }

  @Post("/")
  @Schema<{ name: string }>((body) => typeof body?.name === "string")
  async post(req: KRQ<{ name: string }>): KIRT {
    return { body: req.body };
  }
}

const app = new Kaito({
  controllers: [new Home()],
}).listen(8080);
```
